# 无光之影的传说

那里是永恒的阴影世界。被极夜侵袭的街道尽头，有一座伫立着的尖塔。爬过那座塔的人，可以去光的世界旅行。但是有传言说那里潜伏着恐怖的恶魔，会啃食来访的旅人的影子，夺走他们的身影。有一天，一位失明的少女来到塔上。少女身患不治之症，生命如风中之烛。

“我想看到最后的光……”

这对生活在阴影世界的人来说是理所当然的愿望。少女爬上螺旋楼梯。这条路感觉是永恒的。

手突然碰到了什么东西。那个很温暖。

“啊，好漂亮啊……”

恶魔对少女的样子感到困惑。以前也曾被这样的目光注视过吗？看不见的眼睛里确实映着光。

“——我将成为你的光。”
